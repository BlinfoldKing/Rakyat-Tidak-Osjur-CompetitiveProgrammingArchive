Magic Boxes

Time Limit : 1 second / test-case
Memory Limit : 32 MB

DESCRIPTION

Mr. Blangkon has N magic boxes. The magic power of each box is that if we write a number in top of the box, the box will produce some sweets as many as the number written in the top of the box. Moreover, if we arrange those boxes in a row successively, the number of sweets is the result of multiplying all numbers from each box. For example, there are four boxes. The numbers 3, 7, 12, 5, are written in top of first box, second box, third box, and fourth box, respectively. Then we will have is 3 x 7 x 12 x 5 = 1260 sweets!

At one time, Mr. Blangkon will go to meet Semar, Gareng, Petruk, and Bagong. Mr. Blangkon wants to bring some sweets to them. Because they are very envious, then Mr. Blangkon wants to bring some sweets so that each of them will get the same amount of sweets and they do not envy with each other.

When Mr. Blangkon saw the boxes he owned, it turned out that the boxes had been arranged in a row and each box had already has a number written on it. Mr. Blangkon can only choose a number of successive boxes in a row and get sweets based on the numbers written in the boxes he chooses.

Your task is to determine how many possible choices of boxes chosen by Pak Blangkon so that Semar, Gareng, Petruk, and Bagong, do not envy with each other.

INPUT FORMAT

The first line consists of an integer N that states number of boxes. The next line consists of N integers Ai that states the number written in ith box.

OUTPUT FORMAT

Write a number that states the number of possibilities.

INPUT EXAMPLE

4
3 7 12 5

OUTPUT EXAMPLE

6

EXPLANATION

There are 6 possible choices for Mr. Blangkon.

  • He can choose only 3rd box and he will get 12 sweets.
  • He can choose 2nd and 3rd box and he will get 84 sweets.
  • He can choose 3rd and 4th box and he will get 60 sweets.
  • He can choose 1st, 2nd, and 3rd box and he will get 252 sweets.
  • He can choose 2nd, 3rd, and 4th box and he will get 420 sweets.
  • He can choose 1st, 2nd, 3rd, and 4th box and he will get 1260 sweets.

CONSTRAINTS

  • 1 ≤ N ≤ 100000
  • 1 ≤ Ai ≤ 1000
