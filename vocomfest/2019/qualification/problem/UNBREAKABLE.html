UNBREAKABLE

Time Limit : 1 second / test-case
Memory Limit : 64 MB

DESCRIPTION

A security guard named David Dunn is coming home from work, when the train he is on is suddenly derailed, the train crashes, and he is the sole survivor, out of 132 people. Without a scratch on his body. - Unbreakable (2000).

Mr. Blangkon believes that there is someone who can be called The Unbreakable. A kind of person who is born with immunity above the average human in general. Mr. Blangkon analyzed several train accidents that have occurred, help Mr. Blangkon to find out whether there is The Unbreakable among the passengers.

You will be given two arrays of integers, A and B, which indicate the health condition of the passengers before and after the accident. The i-th value of the array shows the level of health of passengers who are in the i-th position. After an accident, the passenger position may be changed and not the same as the position before the accident. Determine whether there is The Unbreakable among the passengers. The Unbreakable is a person that has the same health level before and after the accident.

INPUT FORMAT

The first line of the input contains an integer T, denoting the number of cases. Each case consist of three line :

  • The first line contains an integers N, the number of the passenger.
  • The second line contains N integers (a1, a2, …, aN) denoting the health level of the ith passenger before the accident.
  • The third line contains N integers (b1, b2, …, bN) denoting the health level of the ith passenger after the accident.

OUTPUT FORMAT

For each case, print “EXIST” if there is at least one the Unbreakable among the passengers. Print “MAY EXIST” if there is a possibility (not 100%) that the Unbreakable is exist. Print “NOT EXIST” if you pretty sure that there are no possibility the Unbreakebale exist among the passengers.

INPUT EXAMPLE

3
5
3 3 3 3 3
1 0 0 0 1
5
2 3 1 4 1
0 0 0 1 0
5
2 3 1 4 1
1 1 0 1 1

OUTPUT EXAMPLE

Case #1: NOT EXIST
Case #2: MAY EXIST
Case #3: EXIST

CONSTRAINTS

  • 1 ≤ T ≤ 20
  • 1 ≤ N ≤ 104
  • 1 ≤ ai ≤ 100
  • 0 ≤ bi ≤ 100
