Joy Strings

Time Limit : 1 second / test-case
Memory Limit : 32 MB

DESCRIPTION

Mr. Blankon likes joy strings. A string is called joy when the string contains all vowels, like 'a', 'e', 'i', 'o', and 'u'.

One day, Mr. Blankon was given a string called str. He wants to know how many joy substrings are contained in str. Help him!

INPUT FORMAT

One line contains str, where each character str is lowercase.

OUTPUT FORMAT

The sum of all cheerful substrings that might match the description above.

1st INPUT EXAMPLE

oceriaku

1st OUTPUT EXAMPLE

1

2nd INPUT EXAMPLE

ceriakuo

2nd OUTPUT EXAMPLE

2

CONSTRAINTS

  • 1 ≤ |str| ≤ 200000
