Reverse %28%29s

Time Limit : 3 second / test-case
Memory Limit : 128 MB

DESCRIPTION

Blangkon is currently learning about programming fundamentals on online platform learning. He met a problem he found challenging called "Reverse parentheses", as you know it's just reverse every character which wrapped in parentheses. Blangkon loves to share anything he find challenging to his friend, Gareng. But he knows Gareng is much smarter than him, so Blangkon decide to tweak the task before share it with Gareng by change the input into URL-encoded format string.

URL-encoded format will change few characters to a new one that browser could accept, but it's too much of them, so Blangkon decide to take some of them, as shown bellow:

(space) %20
( %28
) %29
, %2C
“ %22
‘ %27
! %21
? %3F

Imagine yourself as Gareng, your task is decode the input and reverse all characters in parentheses.

Input string will always be well-formed with matching “%28%29”s.

INPUT FORMAT

Given nonempty alphanumeric string S with URL encoded format. It is guaranteed that all parentheses in S form a regular bracket sequence.

OUTPUT FORMAT

The decoded format of string S with all characters in parentheses reversed.

1st INPUT EXAMPLE

%28abcde%29

1st OUTPUT EXAMPLE

edcba

2nd INPUT EXAMPLE

Gu%28deg%29%20%28Yu%20%28Djum%29%29

2nd OUTPUT EXAMPLE

Guged Djum uY

3rd INPUT EXAMPLE

100%%20%28%28halal%29%29%21

3rd OUTPUT EXAMPLE

100% halal!

EXPLANATION

  • For first test case %28abcde%29 is decoded into (abcde), thus Gareng will reverse the input into edcba.
  • For the second test case, Gu%28deg%29%20%28Yu%20%28Djum%29%29 is decoded into Gu(deg) (Yu (Djum)), thus the answer is Guged Djum Uy.

CONSTRAINTS

  • 1 ≤ S ≤ 1000
